# Spalilsya-Podelilsya – Mentor-based Recommendations

## Что это?
Концепция «Спалился-поделился» – это механизм, который позволяет пользователю **скопировать настройки рекомендаций** другого пользователя (ментора). Проект исследует, как различные рекомендательные архитектуры (LightGCN, NGCF, Social GNN, MARCO, HSTU и др.) могут быть адаптированы для передачи вкуса через граф, контрастное обучение или динамическое смешивание эмбеддингов.

## Мотивация
Стандартные CF-модели не учитывают влияния других пользователей явно. Менторский подход:
- помогает холодным пользователям (один ментор заменяет 20–30 собственных взаимодействий);
- расширяет разнообразие рекомендаций, снижая «фильтрующий пузырь»;
- добавляет интерпретируемости («этот товар понравился вашему ментору»).
